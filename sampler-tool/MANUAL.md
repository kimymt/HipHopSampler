# Hip Hop Sampler — 操作マニュアル

初めての方向けの操作ガイドです。**5分で1つのビート** が作れることを目標に書いています。

---

## ⚡ 30秒クイックスタート

1. **空のパッド** をクリックして選択（青い枠が出る）
2. その状態で **音声ファイル(WAV/MP3)** を画面にドラッグ&ドロップ
3. パッドがオレンジになったら成功 → クリックで音が鳴る
4. 下の **シーケンサー** の四角をクリックして点灯させる
5. 上の **▶（再生ボタン）** を押す → 自動的にビートが鳴る

これだけです。詳細は以下を読んでください。

---

## 🎛 画面の見方

```
┌─────────────────────────────────────────────────┐
│  ① ヘッダー (BPM, 再生, 録音, ステータス)          │
├──────────────────────────┬──────────────────────┤
│                          │                      │
│  ② パッドグリッド         │   ③ サンプル編集      │
│   (音を鳴らす場所)         │     (波形 + 設定)    │
│                          │                      │
├──────────────────────────┴──────────────────────┤
│                                                  │
│  ④ シーケンサー (16ステップで自動再生パターン)      │
│                                                  │
└─────────────────────────────────────────────────┘
```

### ① ヘッダー（一番上）

| 表示 | 意味 |
|------|------|
| ▶ ボタン | **再生 / 停止**。シーケンサーを動かす |
| ● ボタン | **録音モード**。再生中にパッドを叩いた瞬間がパターンに記録される |
| `BPM` | テンポ（1分間の拍数）。クリックで数字編集可。Hip Hopは 85〜95 が定番 |
| `PADS x/16` | 何個サンプルを読み込んだか |
| `IDLE / PLAYING` | 再生中かどうか |

### ② パッドグリッド（左、4×4の16マス）

各マス（パッド）に音声ファイルを割り当てて、クリックやキーボードで鳴らします。

- **右上の英字** = キーボードのショートカット（`Q`/`W`/`E` など）
- **左上の番号** = パッド番号（01〜16）
- 状態:
  - **クリーム色（DROP表示）** = 空。何も入っていない
  - **オレンジ** = サンプル読み込み済み
  - **青枠** = 現在選択中（編集対象）
  - **黄色フラッシュ** = 押下中

### ③ サンプル編集パネル（右）

選択中のパッドに割り当てたサンプルの **波形表示** と **音量・パン調整**。
パッドが選択されていない時は `SELECT A PAD` と表示されます。

- **黄色いマーカー** = 再生開始/終了点。**ドラッグして編集** 可能
- **VOLUME** スライダー = 音量
- **PAN** スライダー = 左右の定位 (-100=左、0=中央、+100=右)
- **×ボタン** = サンプル削除

### ④ シーケンサー（一番下、16マス）

選択中のパッドのステップシーケンス（自動再生パターン）。

- **マスをクリック** = そのタイミングでパッドが鳴る
- 4マス毎に **薄い色分け** = 1拍 = 4分音符。1, 5, 9, 13 が拍頭
- **黄色のカーソル** = 再生中の位置
- **CLEAR** ボタン = このパッドのパターンを全消去

## 🥁 ビートを作る基本フロー

### ステップ1: サンプルを読み込む

1. 鳴らしたいパッドを **クリック**（青枠が出る）
2. 音声ファイル（WAV / MP3）をブラウザにドラッグ&ドロップ
3. パッドがオレンジに変わる

> 💡 **重要**: ドロップする前に必ず **パッドを選択** してください。選択していないと無視されます。
> 別のパッドにロードしたい時は、別パッドを選択してから次のファイルをドロップ。

### ステップ2: 単発で音を確認

- **クリック** または **キーボードショートカット** で鳴らす
- 例: 1行目左上のパッドは `1` キー、2行目左は `Q` キー、3行目左は `A` キー、4行目左は `Z` キー

### ステップ3: パターンを組む（基本キックパターン例）

1. キックを入れたいパッドを選択（オレンジになっていることを確認）
2. シーケンサーの **1, 5, 9, 13** マスをクリック → 4-on-the-floor のキックパターン
3. ▶ Play を押す → 自動再生開始

### ステップ4: ハイハットを重ねる

1. 別のパッドにハイハット音をロード
2. そのパッドを選択
3. シーケンサーで **3, 7, 11, 15** をクリック → 裏拍ハイハット
4. 再生中に変更してもOK（リアルタイム反映）

### ステップ5: スネアを足す

1. スネア用のパッドを選択
2. シーケンサーで **5, 13** をクリック → 2拍4拍にスネア

これでドラム3パターンの基本ビートが完成。BPMを変えると即座にテンポが変わります。

## 🎙 ライブ録音機能（応用）

シーケンサーをポチポチ打つ以外に、**演奏してパターンを記録** することもできます。

1. ● **Record** ボタンをクリック（赤くパルス）
2. ▶ **Play** を押す（再生開始）
3. **再生中にパッドを叩く** → 鳴らしたタイミングがそのパッドのパターンに記録される
4. もう一度 ● を押すと録音終了

## ⌨ キーボードマッピング

```
[ 1 ] [ 2 ] [ 3 ] [ 4 ]   ← パッド 01〜04
[ Q ] [ W ] [ E ] [ R ]   ← パッド 05〜08
[ A ] [ S ] [ D ] [ F ]   ← パッド 09〜12
[ Z ] [ X ] [ C ] [ V ]   ← パッド 13〜16
```

各パッドの右上にショートカットキーが書かれています。

## 🛠 サンプルの編集

### 音量・パン調整
1. パッドを選択
2. 右側の **VOLUME / PAN** スライダーで調整
3. 即座に反映される（変更は保存される）

### トリミング（不要部分のカット）
1. 波形ディスプレイの **黄色マーカー** を見る（左右に2本）
2. 左マーカーをドラッグ → 開始位置を変える
3. 右マーカーをドラッグ → 終了位置を変える
4. トリム範囲が薄黄色でハイライトされる

### サンプル削除
- 右側パネルの **× ボタン** をクリック

---

## 💾 セッションの保存

- **BPM** と **シーケンサーパターン** はブラウザを閉じても自動保存されます（localStorage）
- 次回開いた時は同じパターンが復元されます

⚠️ **注意**: 音声ファイル自体は保存されないので、ブラウザリロード後は再ドロップが必要です。

## ❓ よくあるトラブル

### Q. ファイルをドロップしてもパッドに入らない
→ **先にパッドをクリックして選択**（青枠が出る状態）してから、ファイルをドロップしてください。

### Q. パッドを叩いても音が鳴らない
- パッドがオレンジになっていますか？（オレンジ = サンプル読み込み済み）
- ブラウザがミュートになっていませんか？
- 初回はブラウザのオーディオ許可が必要なことがあります（最初のクリックで起動）

### Q. ▶ Play を押してもシーケンサーが進まない
→ シーケンサーのマスがどこか1つでも点灯していますか？ 全部消えている状態だと「鳴らないビート」が再生されます。

### Q. 別のパッドのシーケンサーパターンを編集したい
→ 編集したいパッドを **クリックして選択** してください。シーケンサーは「現在選択中のパッド」のパターンを表示しています。
パッドが空でも青枠選択はできます（パターンだけ作っておく）。

### Q. キーボードショートカットが効かない
→ 文字入力フィールド（BPM入力など）にフォーカスがあると無効化されます。一度別の場所をクリックしてください。

### Q. 全部最初からやり直したい
→ ブラウザの開発者ツール → Application → Local Storage → `sampler.session.v1` を削除 → リロード。
（後日 RESET ボタンを追加予定）

## 📋 用語ミニ辞典

| 用語 | 意味 |
|------|------|
| **BPM** | Beats Per Minute。1分間に拍が何回あるか |
| **ステップ** | シーケンサーの1マス。16ステップ = 4拍 = 1小節 |
| **パターン** | 1パッドあたりの16ステップの並び |
| **トリミング** | サンプルの不要部分（前後の無音や息継ぎなど）を切る作業 |
| **パン** | 音の左右定位。ハイハットを少し右に振るなどステレオ感を出す |
| **4-on-the-floor** | 1, 5, 9, 13 にキックを入れる、ダンスミュージック定番のキックパターン |
