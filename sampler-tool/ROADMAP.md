# Hip Hop Sampler — ロードマップ

調査結果と初回UIレビューから抽出した改善項目。優先度別に整理。

## 🔴 P0 — 初心者UX致命傷 (今フェーズで対応)

- [x] **パッド数を4x4=16に変更** — MPC標準、情報量を抑える
- [ ] **キーボードキー表示** — 各パッドに割当キーを視覚表示
- [ ] **空パッドの「Drop here」案内** — 初見ユーザーの操作起点を明示
- [ ] **ステータス表示** — ロード数、再生中パッドのハイライト
- [ ] **トランスポートバー** — BPM、Play/Stop、Record
- [ ] **ビジュアルの「Hip Hop感」** — テクスチャ、ネオングロー、フォント

- [ ] **16ステップシーケンサー** — 選択パッドのパターン入力
- [ ] **ミキサーパネル** — パッド毎のVolume/Pan
- [ ] **波形トリミング** — ドラッグで開始/終了点を編集
- [ ] **ループ設定** — ループ開始/終了ポイント、自動検出
- [ ] **パターン保存・復元** — IndexedDBでセッション永続化

- [ ] **エフェクト** — リバーブ、ディレイ、ローパスフィルタ
